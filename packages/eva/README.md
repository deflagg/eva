# Eva (TypeScript daemon)

## Prerequisites

- `nvm`
- Node.js latest stable (Current)

## Setup

```bash
nvm install node
nvm use node
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment

- `EVA_PORT` (default `8787`)
- `QUICKVISION_WS_URL` (default `ws://localhost:8000/infer`)
