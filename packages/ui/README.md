# Eva UI (Vite + React + TypeScript)

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Default URL: `http://localhost:5173`

## Build

```bash
npm run build
```

## Environment

- `VITE_EVA_WS_URL` (default `ws://localhost:8787/eye`)

## Iteration 2 behavior

- Connects to Eva WebSocket endpoint.
- Shows connection status.
- Displays a log panel of incoming/outgoing events.
- Includes a **Send test message** button to verify echo behavior.
